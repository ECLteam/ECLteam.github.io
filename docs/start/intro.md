## ::streamline-ultimate-color:copy-paste-1::项目介绍

==EuoraCraft Launcher== 是一个专为 Minecraft 设计的**第三方**启动器 ，旨在为玩家提供更**便捷**的游戏体验。它支持多版本和模组，允许用户轻松管理和切换不同的版本。

同时，启动器使用 `Python` 编写，支持跨平台运行
